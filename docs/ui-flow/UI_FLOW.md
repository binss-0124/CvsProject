# UI/UX 플로우 다이어그램

## 🎨 UI/UX 설계 원칙

### 핵심 원칙
1. **사용자 중심 설계**: 각 역할별 사용자의 니즈와 행동 패턴 반영
2. **직관적 네비게이션**: 최소 클릭으로 원하는 기능에 접근
3. **일관된 디자인**: 전체 시스템에서 통일된 UI 컴포넌트 사용
4. **반응형 디자인**: 모바일과 데스크톱에서 최적화된 경험
5. **접근성**: 모든 사용자가 쉽게 이용할 수 있는 인터페이스

### 디자인 시스템
- **컬러 팔레트**: Primary Blue (#3B82F6), Success Green (#10B981), Warning Orange (#F59E0B), Error Red (#EF4444)
- **타이포그래피**: Inter 폰트 패밀리 사용
- **간격**: 4px 단위 기반 그리드 시스템
- **그림자**: 계층적 깊이 표현을 위한 섀도우 시스템

## 📱 고객 앱 UI 플로우

### 메인 플로우: 주문 과정

```
[시작] → [로그인/회원가입] → [매장 선택] → [상품 선택] → [장바구니] → [주문 확인] → [결제] → [완료]
```

#### 1. 인증 플로우 (Authentication Flow)

```
스플래시 화면
      ↓
로그인 여부 확인
    ↙     ↘
[기존 사용자]  [신규 사용자]
    ↓           ↓
로그인 화면    회원가입 화면
    ↓           ↓
    ↓       이용약관 동의
    ↓           ↓
    ↓       개인정보 입력
    ↓           ↓
    ↓       이메일 인증
    ↓           ↓
    └─────→ 메인 대시보드
```

**주요 화면 구성:**
- **스플래시 화면**: 브랜드 로고, 로딩 인디케이터
- **로그인 화면**: 이메일/비밀번호 입력, 소셜 로그인 옵션, 비밀번호 찾기
- **회원가입 화면**: 단계별 폼 (개인정보 → 약관동의 → 인증)

#### 2. 매장 선택 플로우 (Store Selection Flow)

```
메인 대시보드
      ↓
위치 권한 요청
      ↓
현재 위치 기반 매장 목록
      ↓
매장 선택 (지도/리스트 뷰)
    ↙     ↘
[픽업]    [배송]
  ↓        ↓
매장 상세  배송 주소 확인
  ↓        ↓
  └────→ 상품 목록
```

**주요 화면 구성:**
- **매장 목록**: 지도 뷰/리스트 뷰 토글, 거리순/평점순 정렬
- **매장 상세**: 매장 정보, 운영시간, 리뷰, 연락처
- **지도 화면**: 네이버 지도 연동, 매장 위치 마커

#### 3. 상품 선택 플로우 (Product Selection Flow)

```
상품 목록 (카테고리별)
      ↓
상품 검색/필터링
      ↓
상품 상세 보기
      ↓
수량 선택 & 옵션 선택
      ↓
장바구니 담기 확인
    ↙     ↘
[계속 쇼핑]  [장바구니 보기]
    ↓           ↓
상품 목록      장바구니 화면
```

**주요 화면 구성:**
- **상품 목록**: 그리드/리스트 뷰, 카테고리 탭, 검색바
- **상품 상세**: 이미지 갤러리, 상품 정보, 리뷰, 관련 상품
- **장바구니**: 상품 목록, 수량 조절, 총액 계산

#### 4. 주문 및 결제 플로우 (Order & Payment Flow)

```
장바구니 화면
      ↓
주문 방식 선택 (픽업/배송)
    ↙     ↘
[픽업]    [배송]
  ↓        ↓
픽업 시간   배송 주소
선택        입력/선택
  ↓        ↓
  └────→ 결제 방법 선택
           ↓
      쿠폰/포인트 적용
           ↓
      최종 주문 확인
           ↓
      토스페이먼츠 결제
           ↓
      결제 완료 화면
           ↓
      주문 상태 추적
```

**주요 화면 구성:**
- **주문 확인**: 주문 상품 목록, 배송 정보, 결제 금액 요약
- **결제 화면**: 결제 방법 선택, 카드 정보 입력, 할인 적용
- **완료 화면**: 주문 번호, 예상 시간, 매장 정보

#### 5. 주문 추적 플로우 (Order Tracking Flow)

```
주문 내역 화면
      ↓
특정 주문 선택
      ↓
주문 상세 정보
      ↓
실시간 상태 업데이트
  ↙     ↓     ↘
[접수]  [준비중] [완료]
  ↓     ↓      ↓
알림   진행률   픽업/배송
받기   표시     완료
```

**주요 화면 구성:**
- **주문 내역**: 진행중/완료된 주문 탭, 주문 상태별 필터
- **주문 상세**: 주문 정보, 상태 타임라인, 매장 연락처
- **상태 추적**: 프로그레스 바, 실시간 업데이트, 알림

## 🏪 점주 앱 UI 플로우

### 메인 플로우: 주문 관리 과정

```
[로그인] → [대시보드] → [주문 관리] → [주문 처리] → [재고 관리] → [매출 분석]
```

#### 1. 점주 대시보드 플로우 (Store Dashboard Flow)

```
점주 로그인
    ↓
대시보드 메인
    ↓
실시간 현황 표시
  ↙   ↓   ↘
[주문]  [재고] [매출]
관리   관리   분석
  ↓    ↓     ↓
상세   상세   상세
화면   화면   화면
```

**주요 화면 구성:**
- **대시보드**: KPI 카드, 오늘 주문 현황, 재고 알림, 매출 차트
- **메뉴 네비게이션**: 주문 관리, 재고 관리, 매출 분석, 설정
- **상태 인디케이터**: 매장 운영 상태, 알림 뱃지

#### 2. 주문 관리 플로우 (Order Management Flow)

```
주문 관리 메뉴
      ↓
주문 목록 (상태별 탭)
  ↙   ↓   ↓   ↘
[대기] [진행] [완료] [취소]
  ↓    ↓    ↓    ↓
주문   주문  주문  주문
상세   상세  상세  상세
  ↓
상태 변경 액션
  ↙   ↓   ↘
[승인] [준비] [완료]
      중    처리
```

**주요 화면 구성:**
- **주문 목록**: 상태별 탭, 시간순 정렬, 검색 기능
- **주문 상세**: 고객 정보, 상품 목록, 특이사항, 연락처
- **상태 변경**: 버튼 기반 액션, 예상 시간 입력, 메모 작성

#### 3. 재고 관리 플로우 (Inventory Management Flow)

```
재고 관리 메뉴
      ↓
상품 목록 (카테고리별)
      ↓
재고 현황 표시
  ↙   ↓   ↘
[부족] [정상] [과다]
  ↓    ↓    ↓
재고   가격  할인
보충   조정  설정
  ↓
물류 요청 생성
```

**주요 화면 구성:**
- **재고 목록**: 상품 검색, 재고 상태 필터, 정렬 옵션
- **재고 상세**: 현재 재고, 최소 재고, 판매 통계
- **재고 조정**: 수량 입력, 조정 사유, 일괄 처리

#### 4. 매출 분석 플로우 (Sales Analytics Flow)

```
매출 분석 메뉴
      ↓
기간 선택 (일/주/월)
      ↓
매출 대시보드
  ↙   ↓   ↘
[총매출] [상품별] [시간대별]
차트    순위     패턴
  ↓     ↓      ↓
상세    상세    상세
분석    분석    분석
```

**주요 화면 구성:**
- **매출 대시보드**: 총 매출, 주문 수, 평균 주문가, 성장률
- **차트 뷰**: 라인 차트, 바 차트, 파이 차트
- **상세 분석**: 드릴다운, 필터링, 데이터 내보내기

## 🏢 본사 관리자 UI 플로우

### 메인 플로우: 전체 관리 과정

```
[로그인] → [통합 대시보드] → [매장 관리] → [물류 관리] → [분석 리포트]
```

#### 1. 통합 대시보드 플로우 (HQ Dashboard Flow)

```
관리자 로그인
      ↓
통합 대시보드
      ↓
전체 현황 모니터링
  ↙   ↓   ↓   ↘
[매장] [주문] [매출] [재고]
현황   현황   현황   현황
  ↓    ↓    ↓    ↓
상세   상세  상세  상세
관리   관리  분석  관리
```

**주요 화면 구성:**
- **통합 대시보드**: 전체 KPI, 지역별 현황, 알림 센터
- **실시간 모니터링**: 주문량, 매출, 시스템 상태
- **이상 징후 알림**: 매출 급락, 시스템 오류, 고객 불만

#### 2. 매장 승인 플로우 (Store Approval Flow)

```
매장 관리 메뉴
      ↓
승인 대기 목록
      ↓
신청서 상세 검토
  ↙   ↓   ↘
[서류] [위치] [사업]
검증   확인   계획
  ↓    ↓    ↓
  └────┼────┘
       ↓
승인/거부 결정
  ↙     ↘
[승인]   [거부]
완료     사유작성
```

**주요 화면 구성:**
- **승인 대기 목록**: 신청일순, 우선순위별 정렬
- **신청서 상세**: 점주 정보, 매장 정보, 첨부 서류
- **검토 도구**: 서류 확대, 메모 작성, 체크리스트

#### 3. 물류 관리 플로우 (Logistics Management Flow)

```
물류 관리 메뉴
      ↓
요청 목록 (우선순위별)
      ↓
요청 상세 검토
  ↙   ↓   ↘
[재고] [배송] [비용]
확인   일정   계산
  ↓    ↓    ↓
  └────┼────┘
       ↓
승인 및 배송 계획
       ↓
진행 상황 추적
```

**주요 화면 구성:**
- **요청 목록**: 긴급도별 색상 코딩, 매장별 필터
- **요청 상세**: 요청 상품, 수량, 예상 비용
- **배송 관리**: 배송 일정, 차량 배정, 진행 상황

## 📱 반응형 디자인 가이드

### 모바일 우선 설계 (Mobile First)

#### 화면 크기별 브레이크포인트
- **모바일**: 320px - 768px
- **태블릿**: 768px - 1024px  
- **데스크톱**: 1024px 이상

#### 모바일 최적화
```
모바일 화면 (375px 기준)
┌─────────────────────────┐
│ [햄버거] 타이틀 [알림]  │ ← 헤더 (60px)
├─────────────────────────┤
│                         │
│    메인 콘텐츠 영역     │ ← 스크롤 가능
│                         │
├─────────────────────────┤
│ [홈][주문][내역][더보기]│ ← 탭 바 (80px)
└─────────────────────────┘
```

#### 태블릿 최적화
```
태블릿 화면 (768px 기준)
┌───────────────────────────────────────┐
│ 로고    네비게이션 메뉴    사용자 정보 │ ← 헤더
├─────────────┬─────────────────────────┤
│             │                         │
│  사이드바   │    메인 콘텐츠 영역     │
│  메뉴       │                         │
│             │                         │
└─────────────┴─────────────────────────┘
```

#### 데스크톱 최적화
```
데스크톱 화면 (1200px 기준)
┌─────────────────────────────────────────────────┐
│ 로고      네비게이션 메뉴        사용자 정보    │ ← 헤더
├─────────┬─────────────────────┬─────────────────┤
│         │                     │                 │
│사이드바 │   메인 콘텐츠 영역  │   보조 패널     │
│메뉴     │                     │   (알림, 위젯)  │
│         │                     │                 │
└─────────┴─────────────────────┴─────────────────┘
```

## 🎨 인터랙션 디자인

### 상태 변화 애니메이션

#### 로딩 상태
```
컴포넌트별 로딩 전략:
• 버튼: 스피너 + 텍스트 변경
• 리스트: 스켈레톤 UI
• 이미지: 블러 → 선명
• 페이지: 프로그레스 바
```

#### 상태 전환
```
주문 상태 변경 플로우:
[대기] --fade--> [진행] --slide--> [완료]
  ↓              ↓              ↓
 회색           주황색         녹색
```

### 사용자 피드백

#### 성공 액션
- **색상**: Green (#10B981)
- **애니메이션**: 체크마크 + 페이드인
- **메시지**: "성공적으로 처리되었습니다"

#### 오류 처리
- **색상**: Red (#EF4444)  
- **애니메이션**: 셰이크 + 하이라이트
- **메시지**: 구체적인 오류 내용 + 해결 방안

#### 경고 알림
- **색상**: Orange (#F59E0B)
- **애니메이션**: 펄스 + 글로우
- **메시지**: 주의사항 + 권장 액션
