# UI 플로우 명세서

## 🎯 UI 플로우 개요

편의점 종합 솔루션의 사용자 인터페이스는 **3가지 역할별 앱**으로 구성되며, 각각 최적화된 플로우를 제공합니다.

## 👤 고객 앱 UI 플로우

### 1. 인증 플로우

```
시작 → 스플래시 화면
  ↓
로그인 상태 확인
  ├─ 로그인됨 → 메인 홈
  └─ 로그인 안됨 → 로그인/회원가입 선택
      ├─ 로그인 → 이메일/비밀번호 입력 → 인증 → 메인 홈
      └─ 회원가입 → 개인정보 입력 → 이메일 인증 → 환영 메시지 → 메인 홈
```

### 2. 메인 홈 플로우

```
메인 홈
├─ 헤더
│   ├─ 알림 아이콘 (빨간 뱃지) → 알림 목록
│   ├─ 장바구니 아이콘 (상품 개수) → 장바구니 화면
│   └─ 프로필 아이콘 → 마이페이지
├─ 검색바 → 검색 결과 화면
├─ 배너 슬라이더 (프로모션)
├─ 빠른 카테고리 (8개 아이콘)
├─ 주변 편의점 (지도 미리보기) → 편의점 목록
├─ 추천 상품 (개인화)
└─ 최근 주문 → 주문 내역
```

### 3. 편의점 선택 및 상품 주문 플로우

```
편의점 목록
├─ 지도 뷰 / 리스트 뷰 토글
├─ 필터 (거리, 영업중, 평점)
├─ 정렬 (거리순, 평점순, 리뷰순)
└─ 편의점 카드 선택
    ↓
편의점 상세 페이지
├─ 편의점 정보 (영업시간, 전화번호, 주소)
├─ 카테고리 탭 (음료, 과자, 라면 등)
├─ 상품 목록 (그리드 뷰)
├─ 검색 및 필터
└─ 상품 카드 선택
    ↓
상품 상세 모달
├─ 상품 이미지 슬라이더
├─ 상품 정보 (가격, 할인, 재고)
├─ 수량 선택 (+/- 버튼)
├─ 옵션 선택 (있는 경우)
└─ 장바구니 담기 / 바로 주문
    ↓
장바구니 화면
├─ 상품 목록 (수량 변경, 삭제 가능)
├─ 쿠폰 적용
├─ 포인트 사용
├─ 주문 옵션 (픽업/배송)
├─ 총 금액 계산
└─ 주문하기 버튼
    ↓
주문서 작성
├─ 배송 정보 (배송 선택 시)
├─ 픽업 시간 (픽업 선택 시)
├─ 요청사항
├─ 최종 금액 확인
└─ 결제하기
    ↓
결제 화면 (토스페이먼츠)
├─ 결제 수단 선택
├─ 카드 정보 입력
└─ 결제 완료
    ↓
주문 완료 화면
├─ 주문번호 표시
├─ 예상 준비 시간
├─ 주문 추적 버튼
└─ 홈으로 가기
```

### 4. 주문 추적 플로우

```
주문 추적 화면
├─ 실시간 상태 표시
│   ├─ 주문 접수 ●○○○
│   ├─ 준비 중 ●●○○
│   ├─ 픽업 대기 ●●●○
│   └─ 완료 ●●●●
├─ 편의점 정보 및 연락처
├─ 주문 상세 내역
├─ 예상 시간 표시
└─ 주문 취소 (접수 단계에서만)
```

### 5. 마이페이지 플로우

```
마이페이지
├─ 프로필 정보
├─ 주문 내역
├─ 포인트 및 쿠폰
├─ 위시리스트
├─ 설정
│   ├─ 개인정보 수정
│   ├─ 알림 설정
│   ├─ 결제 수단 관리
│   └─ 로그아웃
└─ 고객센터
```

## 🏪 점주 앱 UI 플로우

### 1. 로그인 및 대시보드

```
점주 로그인
├─ 이메일/비밀번호 입력
├─ 매장 선택 (여러 매장 운영 시)
└─ 인증 완료
    ↓
점주 대시보드
├─ 오늘의 매출 (실시간)
├─ 신규 주문 알림 (실시간 카운터)
├─ 재고 부족 알림
├─ 빠른 액션 버튼
│   ├─ 새 주문 확인
│   ├─ 재고 관리
│   └─ 매출 분석
└─ 최근 활동 요약
```

### 2. 주문 관리 플로우

```
주문 관리 화면
├─ 탭 메뉴
│   ├─ 대기 중 (빨간 뱃지)
│   ├─ 준비 중
│   ├─ 픽업 대기
│   └─ 완료
├─ 주문 카드 (시간순 정렬)
│   ├─ 주문 정보 요약
│   ├─ 상태 변경 버튼
│   └─ 상세보기
└─ 주문 상세 모달
    ├─ 고객 정보
    ├─ 주문 내역
    ├─ 특별 요청사항
    ├─ 상태 변경 (드롭다운)
    └─ 저장/알림 전송
```

### 3. 재고 관리 플로우

```
재고 관리 화면
├─ 검색 및 필터
│   ├─ 카테고리별 필터
│   ├─ 재고 상태 (품절, 부족, 정상)
│   └─ 상품명 검색
├─ 재고 현황 카드
│   ├─ 상품 이미지 및 정보
│   ├─ 현재 재고량 (색상 구분)
│   ├─ 최소 재고량 설정
│   └─ 빠른 수정 버튼
├─ 대량 수정 모드
└─ 발주 요청 버튼
    ↓
발주 요청 화면
├─ 부족 상품 자동 선택
├─ 수량 조정
├─ 우선순위 설정
├─ 희망 배송일
└─ 요청 제출
```

### 4. 매출 분석 플로우

```
매출 분석 화면
├─ 기간 선택 (오늘/주간/월간)
├─ KPI 카드
│   ├─ 총 매출
│   ├─ 주문 수
│   ├─ 평균 주문 금액
│   └─ 성장률
├─ 매출 차트 (선 그래프)
├─ 인기 상품 순위
├─ 시간대별 매출
└─ 보고서 다운로드
```

```

│   ├─ 매출 분석

│   └─ 고객 분석
```

## 📱 공통 UI 컴포넌트

### 1. 네비게이션

```
하단 탭 네비게이션 (고객)
├─ 홈 (🏠)
├─ 검색 (🔍)
├─ 주문내역 (📋)
├─ 위시리스트 (❤️)
└─ 마이페이지 (👤)

사이드 메뉴 (점주/본사)
├─ 대시보드
├─ 주문 관리
├─ 재고 관리
├─ 분석
├─ 설정
└─ 로그아웃
```

### 2. 상태 및 알림

```
주문 상태 표시
├─ 컬러 코딩 (회색/노랑/파랑/초록)
├─ 아이콘 표시
├─ 프로그레스 바
└─ 시간 표시

실시간 알림
├─ 토스트 메시지 (자동 사라짐)
├─ 푸시 알림 (앱 외부)
├─ 뱃지 카운터 (탭/아이콘)
└─ 인앱 알림 (지속적)
```

### 3. 데이터 로딩

```
로딩 상태
├─ 스켈레톤 UI (콘텐츠 모양)
├─ 스피너 (중앙 정렬)
├─ 프로그레스 바 (파일 업로드)
└─ 페이지네이션 로딩

에러 상태
├─ 에러 메시지 표시
├─ 재시도 버튼
├─ 고객센터 연결
└─ 홈으로 가기
```

## 🔄 실시간 업데이트 플로우

### 1. 주문 상태 업데이트

```
고객 주문 → 점주 화면 실시간 알림 (사운드 + 진동)
점주 상태 변경 → 고객 화면 실시간 업데이트 (푸시 알림)
배송 상태 변경 → 관련 화면 자동 새로고침
```

### 2. 재고 업데이트

```
주문 완료 → 재고 자동 차감 → 점주 재고 화면 업데이트
재고 부족 → 자동 알림 → 발주 제안 팝업
신규 입고 → 재고량 업데이트 → 품절 해제 알림
```

### 3. 알림 플로우

```
이벤트 발생 → 데이터베이스 트리거 → Realtime 채널 → 클라이언트 수신
                                      ↓
                              웹 푸시 알림 (브라우저)
                                      ↓
                              앱 내 알림 (실시간 UI 업데이트)
```

## 📱 반응형 디자인 고려사항

### 1. 화면 크기별 레이아웃

```
모바일 (< 768px)
├─ 세로 레이아웃 중심
├─ 하단 네비게이션
├─ 풀스크린 모달
└─ 스와이프 제스처

태블릿 (768px - 1024px)
├─ 2단 컬럼 레이아웃
├─ 사이드 네비게이션
├─ 슬라이딩 패널
└─ 터치 최적화

데스크톱 (> 1024px)
├─ 3단 컬럼 레이아웃
├─ 고정 사이드바
├─ 오버레이 모달
└─ 마우스 호버 효과
```

### 2. 터치 인터페이스

```
버튼 크기: 최소 44px × 44px
간격: 최소 8px 여백
제스처: 스와이프, 핀치, 탭
피드백: 햅틱, 애니메이션
```
